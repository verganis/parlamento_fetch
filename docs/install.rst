Installation
============

Install libffi-dev

.. code-block:: bash

    sudo apt-get install libffi-dev
    
Install cffi
    
.. code-block:: bash

    sudo pip install cffi

Install project requirements
    
.. code-block:: bash

    pip install -r requirements.txt
    
Create output directory

.. code-block:: bash
    
    mkdir output

Install and setup smtp as explained Here_.
